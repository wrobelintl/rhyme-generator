## 2. Word-frequency ranks — Creative Commons Attribution 3.0 Unported (CC BY 3.0)

Frequency rank data is derived from the **Google Books Ngram Viewer datasets**
(https://books.google.com/ngrams), whose official datasets page states:
"This compilation is licensed under a Creative Commons Attribution 3.0
Unported License."

The ranks were obtained via the pre-computed list `frequency-alpha-alldicts.txt`
from **gwordlist** by hackerb9 (https://github.com/hackerb9/gwordlist), whose
README states: "I'm explicitly releasing the data produced under the same
license as Google granted me: Creative Commons Attribution 3.0." Only that data
file is used; none of the gwordlist scripts (GPL >= 3) are used or included.

**Attribution:** "Contains frequency data derived from the Google Books Ngram
Viewer datasets (https://books.google.com/ngrams), CC BY 3.0, via gwordlist
(https://github.com/hackerb9/gwordlist)."

The derived rank information inside `data/rhymes.min.json` is accordingly
available under **CC BY 3.0**; the CMU-derived pronunciation grouping carries
the BSD-2-Clause notice above.
