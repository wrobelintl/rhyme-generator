# Rhyme data sources & licenses

`data/rhymes.min.json` is generated by `tools/build-rhyme-data.mjs` at build time.
It is **derived data** (pronunciation-based rhyme keys), not a copy of the source
dictionaries. Regenerate with:

```
node tools/build-rhyme-data.mjs
```

## Common-word filter / ranking — google-10000-english
- Source: https://github.com/first20hours/google-10000-english
  (`google-10000-english-usa-no-swears.txt`)
- License: MIT (the compiled list). Derived from the Google Web Trillion Word Corpus.
- Used only to (a) restrict the dataset to common words and (b) rank rhymes by
  frequency. No raw list is redistributed here.

## Notes
- Neither source dictionary is vendored into the repo; both are fetched at build
  time and only the derived `rhymes.min.json` is committed.
- The dataset is intentionally limited to ~common words (spike scope) to keep the
  file small and results free of obscure/proper-noun junk.
